# plan2meet
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### Json server setup
```
npm install -g jason server
```
### Stores data in json
```
json-server --watch db.json
```
or 
```
json-server db.json
```
